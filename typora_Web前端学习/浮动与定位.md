# 浮动与定位

> #### 传统网页布局方式
>
> 在学习浮动之前，先了解传统的网页布局方式
> 网页布局方式有以下五种：
>
> - 标准流（普通流、文档流）：网页按照元素的书写顺序依次排列
> - 浮动
> - 定位
> - `Flexbox`和`Grid`（自适应布局）
>
> `标准流`是由块级元素和行内元素按默认规定的方式来排列，块级就是占一行，行内元素一行放好多个元素。

> #### 浮动
>
> 元素脱离文档流，根据开发者的意愿漂移到网页的任意方向
>
> `浮动`属性用于创建浮动框，将其移动到一边，直到左边缘或右边缘触及包含块或另一个浮动框的边缘，这样即可使得元素进行浮动。
>
> 语法
>
> ```text
> 选择器{
> 	float:left/right/none;
> }
> ```
>
> 注意：浮动是相对于父元素浮动，只会在父元素的内部移动

> #### 浮动的三大特性
>
> - 脱标：脱离标准流
> - 一行显示，顶部对齐
> - 具备行内块元素特性

> 浮动示例：
>
> ```html
> <head>
>     <meta charset="UTF-8">
>     <meta name="viewport" content="width=device-width, initial-scale=1.0">
>     <title>浮动</title>
>     <style>
>         .father{
>             background-color: aqua;
>             /* height: 150px; 将父本高度去除以制造一坍缩父本的例子*/
>             border: 3px solid brown;
>             overflow: hidden;/*清除浮动的指令*/
>         }
>         /* 伪元素选择器来清除坍缩的浮动 */
>         .father::after{
>             content: "";
>             display: table;
>             clear: both;
>         }
>         .left-son{
>             width: 100px;
>             height: 100px;
>             background-color: yellow;
>             float: left;
>         }
>         .right-son{
>             width: 100px;
>             height: 100px;
>             background-color: red;
>             float:right;
>         }
>     </style>
> </head>
> <body>
>     <div class="father">
>         <div class="left-son">左浮动</div>
>         <div class="left-son">左浮动</div>
>         <div class="left-son">左浮动</div>
>         <div class="left-son">左浮动</div>
>         <div class="right-son">右浮动</div>
>     </div>
>     <p>这是一段文本这是一段文本这是一段文本</p>
>     <p>父本出现坍缩父本出现坍缩</p>
> </body>
> ```
>
> 设定了一个父元素和几个子元素，一旦父元素高度出问题，则父本就会出现坍缩，父本下面的文本就会出现问题，需要进行清除坍缩的操作
>
> 清除坍缩的两种操作
> 第一种是在父本控制器中加入`overflow: hidden`属性
>
> ```html
>   <style>
>         .father{
>             background-color: aqua;
>             /* height: 150px; */
>             border: 3px solid brown;
>             overflow: hidden;/*清除浮动的指令*/
>         }
> </style>
> ```
>
> 第二种是用伪元素选择器清理坍缩
>
> ```html
> <style>
> /* 伪元素选择器来清除坍缩的浮动 */
>         .father::after{
>             content: "";
>             display: table;
>             clear: both;
>         }
> </style>
> ```

> #### 定位
>
> 定位布局可以精确定位，但缺乏灵活性
>
> 定位方式：
>
> - 相对定位：相对于元素在文档流中的正常位置进行定位
> - 绝对定位：相对于其最近的已定位祖先元素进行定位，不占据文档流
> - 固定定位：相对于浏览器窗口进行定位。不占据文档流，固定在屏幕上的位置，不随滚动而移动

> 相对定位：就是与原先的位置进行移动
> 示例：
>
> ```html
> <head>
>     <meta charset="UTF-8">
>     <meta name="viewport" content="width=device-width, initial-scale=1.0">
>     <title>定位</title>
>     <style>
>         .box1{
>             height: 350px;
>             background-color: aqua;
>         }
>         .box-normal{
>             width: 100px;
>             height: 100px;
>             background-color: purple;
>         }
>         .box-relative{
>             width: 100px;
>             height: 100px;
>             background-color: pink;
>             /* 相对定位的元素 */
>             position: relative;
>             left: 120px;
>             top: 40px;
>         }
>     </style>
> </head>
> <body>
>     <h1>相对定位</h1>
>     <div class="box1">
>         <div class="box-normal"></div>
>         <div class="box-relative"></div>
>         <div class="box-normal"></div>
>     </div>
> </body>
> ```
>
> `w100+h100`，是长宽的简介输入

> 绝对定位：和相对定位相比，绝对定位会退出文档流，后面的元素会顶替绝对定位的位置，并且是相对于最近的已经定位的父级元素进行定位的，如果没有已经定位的父级元素，就相对于html标签进行定位
>
> ```html
> <head>
>     <meta charset="UTF-8">
>     <meta name="viewport" content="width=device-width, initial-scale=1.0">
>     <title>定位</title>
>     <style>
>    		.box2{
>             height: 350px;
>             background-color: yellow;
>             /*距离网页底部的距离*/
>             margin-bottom: 300px;
>         }
>         .box-absolute{
>             width: 100px;
>             height: 100px;
>             background-color: yellowgreen;
>             position: absolute;
>             left: 120px;
>         }
>     </style>
> </head>
> <body>
>         <h1>绝对定位</h1>
>     <div class="box2">
>         <div class="box-normal"></div>
>         <div class="box-absolute"></div>
>         <div class="box-normal"></div>
>     </div>
> </body>
> ```

> 固定定位：相对于浏览器窗口进行定位，即使页面滚动也会保持在窗口的相同位置
>
> 代码：
>
> ```html
> <head>
>     <meta charset="UTF-8">
>     <meta name="viewport" content="width=device-width, initial-scale=1.0">
>     <title>定位</title>
>     <style>
> /* 固定定位的选择器 */
>         .box-fixed{
>             width: 100px;
>             height: 100px;
>             background-color: bisque;
>             position: fixed;
>             right: 0;
>             top: 300px;
>         }
>     </style>
> </head>
> <body>
>     </div>
>     <h1>固定定位</h1>
>     <div class="box-fixed"></div>
> </body>
> ```
>
